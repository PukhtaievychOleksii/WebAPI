
This project allows you to add various books, users and reservations 
to the system by using appropriate endpoints and requests.

All the neccessary requests for testing are gathered in the folder "requests".
The only thing you need to do is to run an app(through Docker or dotnet,it is up to you) and update
values of environment variables userId,bookId,reservationId located in /.vscode/settings.json

Used technilogies:
ASP.NET,
Visual Studio Code,
Visual Studio Code extenstions: REST Client
Entity Framework Core(Microsoft.EntityFrameworkCore package)
Docker (only Dockerfile)
